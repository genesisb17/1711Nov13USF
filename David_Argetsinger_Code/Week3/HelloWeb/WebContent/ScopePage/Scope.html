Scopes in Java

Scope in java relates to accessibility of variables. Scopes come in four levels:

  • Static/Class
  • Instance
  • Method
  • Block

Static/Class

Static scope can be declared by using the keyword Static. When the Static keyword is used it makes the Static member/block belong to the class rather than a specific instance. Static variables are available without instantiating the class. Static declared variables are class level variables that all instances will use the same memory location. Static can also be applied to blocks {}.

Instance

For a variable to be instance scoped it must be outside of a method and not have the static keyword. Fields of the instance scope are available within the scope of the object. Instance variables must be accessed by an object of a class. Every instance of the object will have fields that will not reference the same memory location.

Method

A field declared within a method have what is known as a method level scope. A method level scope means that the field can only be accessed within the method.
