Procedures

A procedure is a named PL/SQL block.

Syntax for a procedure:


			CREATE [OR REPLACE] PROCEDURE procedure_name
				[ (parameter [,parameter]) ]
			IS 
				[declaration_section]
			BEGIN
				executable_section
			[EXCEPTION
				exception_section]
			END [procedure_name];

Recommend [OR REPLACE] be included always*
There is no difference between IS or AS*

Properties of a procedure:

  • Execute directly
  • INSERT, UPDATE, DELETE
  • Multiple batches of SQL
  • Input parameters
  • Output parameters

Example of stored procedure

Here is a stored procedure that returns info from some columns in a table:


			CREATE OR REPLACE PROCEDURE 
			GET_CUST_INFO(cust_id IN Customer.customerid%TYPE,
						cust_name OUT Customer.firstname%TYPE,
						cust_comp OUT Customer.company%TYPE)
			AS
			f_name Customer.firstname%TYPE;
			l_name Customer.lastname%TYPE;
			BEGIN
			select firstname, lastname into f_name, l_name from
			Customer where customerid=cust_id;
			cust_name:=f_name || ' ' || l_name;
			select company into cust_comp from Customer
			where customerid=cust_id;
			END;

Here is a way to call the stored procedure:


			SET SERVEROUTPUT ON;
			declare
			c_id Customer.customerid%TYPE;
			c_name Customer.firstname%TYPE;
			c_comp Customer.company%TYPE;
			begin
			c_id:=12;
			get_cust_info(cust_id => c_id,
						cust_name => c_name,
						cust_comp => c_comp);
			DBMS_OUTPUT.PUT_LINE(c_name || ' | ' || c_comp);
			end;
