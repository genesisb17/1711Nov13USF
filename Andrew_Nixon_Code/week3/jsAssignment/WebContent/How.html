    How does JBDC does work?

  • Statement: Used to make queries to the database directly.
  • PreparedStatement: Used to make DML statement to the database. The crucial difference between a Statement and a PreparedStatemenr is that a PeparedStatement takes input from the user. This means steps must be ut in place to avoid SQL injection.
  • CallableStatement: Used to call a stored procedure in he database.
  • ResultSet: An interface used to return the results upon executing a statement
  • DriverManager: It is a class used to manage drivers for Databases. It can be considered the connection factory.
  • Connection: An interface that allows access to a session of a database
