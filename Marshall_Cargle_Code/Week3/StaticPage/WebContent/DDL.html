DDL

DDL stands for Data Definition Language

It contains Create, Alter, Drop, Truncate, and Rename

Create
Used to create objects in the database.

Create Table table_name ([Columns_names]) [parameters]
Alter
Used to change the structure of the database.

ALTER TABLE table_name ADD column_name type
Drop
Used to delete objects from the database.

DROP TABLE table_name
Truncate
Used to delete all data inside a table, but not the table itself.
Rename
Used to change the name of an object in the database.
