Getting your money back is easy with five simple steps
After logging in:
  1. Create a request from the logged-in menu on the left
  2. Select the type of reimbursement you are requesting
  3. Put in the amount to be reimbursed
  4. Describe the purpose for the reimbursement
  5. Attach a receipt if you have one available
That's it! Submit the request, and a resolver will get back to you shortly!
