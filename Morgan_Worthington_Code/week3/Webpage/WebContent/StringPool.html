The String Pool is a location of memory where the Java Virtual Machine stores string object literals.

Interning

Since strings are immutable, it is possible to save memory in this pool by making string variables refer to only one literal if they have the same value, as illustrated below. When a string variable is initialized, the string pool is checked to determine if it has the same value as an existing string. If it does, the variable refers to the existing string literal. Otherwise, a new string literal is created.

Garbage Collection
The String Pool is stored in the Heap Space which is garbage collected by the Java Virtual Machine. When a literal in the String Pool has no reference to it, it is set for garbage collection to free up memory.
